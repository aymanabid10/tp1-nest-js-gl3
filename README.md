# TP1 NestJS CV Manager

Simple NestJS API for managing CVs, users, and skills.

## Features

- JWT authentication
- Role-based authorization
- CV ownership enforcement
- Validation and Swagger docs

## Quick Start

1. Install dependencies

corepack pnpm install

2. Create environment file

Copy .env.example to .env and update values.

3. Run the app

corepack pnpm run start:dev

## Useful Commands

- Build: corepack pnpm run build
- Lint: corepack pnpm run lint
- Unit tests: corepack pnpm test
- E2E tests: corepack pnpm test:e2e
- Seed data: corepack pnpm run seed:cvs
